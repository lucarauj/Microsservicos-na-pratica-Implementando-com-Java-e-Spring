# Microsservicos na prática: Implementando com Java e Spring

## Dependências:

- Spring Web
- Spring Data JPA
- PostgreSQL Driver
- Lombok
- Validation
- Flyway Migration
- ModelMapper

## Anotações:

- @Entity
- @Table
- @Getter
- @Setter
- @NoArgsConstructor
- @AllArgsConstructor
- @Id
- @GeneratedValue(strategy = GenerationType.IDENTITY)
- @NotNull
- @Positive
- @NotBlank
- @Size
- @Enumerated(EnumType.STRING)
- @RestController
- RequestMapping
- @Service
- @Autowired
